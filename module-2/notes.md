# Módulo 2 — Funções

Objetivo:
Entender funções como transformação de valores.

O que eu acho que é uma função:
(escreva com suas palavras, mesmo que esteja incompleto ou errado)
é um código criado para processar outros códigos. Ela pode receber variáveis, objetos e até outras funções como parâmetros/ processá-los, extraindo algum significado dele, e retorna alguma resposta que pode ser variáves, objetos, outra função?(não tenho certeza).

Se você escreve let x = minhaFuncao(),
então minhaFuncao precisa retornar algo útil.

Regra de ouro (grava isso)
Se não retorna, não atribua.

Usar variável pra receber função sem retorno é furada, o máximo que ela pode fazer é devolver um log na tela e um undefined por não ter retorno;
