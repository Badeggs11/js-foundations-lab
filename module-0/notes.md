# Módulo 0 — Preparação

Objetivo deste módulo:
Aprender a estudar programação de forma consciente, lenta e sólida.

## Regras pessoais
- Não copiar código pronto
- Escrever antes de rodar
- Prever o resultado antes do console.log
- Errar faz parte

## Como vou estudar
- Exercícios pequenos
- Repetição
- Explicação com minhas palavras
